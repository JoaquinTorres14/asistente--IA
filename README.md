LexiCode AI - Mentor Pro V1.0

Asistente de programación con IA diseñado para ofrecer mentoría técnica y depuración de código en tiempo real.

Características:

Interfaz de Alta Visibilidad: Diseño con estilo Glassmorphism y botones de alto contraste para una mejor usabilidad.

Soporte de Voz: Integración de lectura de texto con velocidad optimizada para una experiencia más fluida.

Modo Híbrido: Permite consultas tanto por voz como por texto manual.

Tecnologías: Desarrollado con JavaScript, Tailwind CSS y desplegado en Vercel.
